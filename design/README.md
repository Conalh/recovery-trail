# design/

Drop zone for Claude Design exports and reference mockups.

Naming: `YYYY-MM-DD-<short-name>.<ext>` — keeps iterations sortable.

Accepted:
- `.html` — Claude Design HTML export
- `.png` / `.jpg` — screenshots
- `.pdf` — PDF export
- `.txt` — share links, notes

Everything in here is version-controlled so the design lineage stays alongside the code.
